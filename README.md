## What I'm Doing

I co-founded **Netanel Systems** — an AI company where every function runs on agents. Engineering, content, QA, delivery — all autonomous. No human in the loop unless something breaks.

Right now I'm deep in:

- **Autonomous agent systems** — multi-agent pipelines that run in production, unsupervised, daily
- **100% AI-native development** — no manual coding. Everything written through natural language and agent collaboration
- **Model philosophy & behavior** — testing how different models reason, where they fail, how to build reliable systems on top of unreliable components
- **Agentic infrastructure** — building the company OS: memory graphs, orchestration, self-improving agents, coordination protocols

This is not prototype work. These systems run every day and get smarter with each cycle.
